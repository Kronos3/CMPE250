\section*{Abstract}
In this laboratory exercise, the functionality of queue structures was investigated. These queues were defined with a record structure and a queue buffer. The buffer was circular meaning when the end of the buffer area was filled, the next item would be enqueued into the start of the queue. The buffer size of the queue was of arbitrary size defined at four bytes.
